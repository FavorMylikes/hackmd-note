    Convert Camera Parameters [usage of ContextCapture Master]

    August 16, 2021 less than 1 minute read

    On this page

    • Convert parameters
      • Model generate
        • 优点
        • 缺点
      • Code
      • Reference

    Convert parameters

    • 拿到空三的参数文件，发现是 Smart3D(soarscape.com) 的软件导出的，尝试下载，发现需要申请
    • 根据关键字查找，找到ContextCapture Master
    • 下载安装

    Model generate

    • Add photos

    • click submit aerotriangulation at general tab.
      • start ContextCapture Engine
      • wait job complish
    • Reconstruction>new Reconstruction
    • click submit new production
      • wait for job complish
      • check production 3D view

    • output
     Model
├── Model.mtl
├── Model.obj
├── Model_0.jpg
├── Model_1.jpg
├── Model_2.jpg
└── Model_3.jpg

    优点

    • 纹理贴图压缩, 避免占用无效面积

    • 非矩形纹理

    • 图片畸变处理， 直接在贴图中使用Undistort的图片

    缺点

    • 阴影被识别为特征点

    • 纹理畸变过度

    Code

    • comera_convert
    • 转换公式，参考Context capture master， 文档
    • xml文件含义
