    Markdown confict with katex

    Two method for rendering katex

    1. Render at server side
      • Jekyll 中使用 KaTeX SsKatex
      • Not sure it could works or not.
      • You need set config at _config.yml
           kramdown:
         math_engine: null
      • The math_engine be set mathjax as default
      • If you wanna use javascript for rendering, you’d better set it be null
      • For more selection for it, check here.

    2. Render at client side -. Require you have katex.js and katex.css at head, katex will render when you brower open. -. So you need consider about the order of rendering markdown and katex -. For example, you have a md file like

       $ABC*$123$DE*F$

      -. It will convert $123$DE$ as Italic first -. And then the rendering for katex
